# Business Insight Pipeline (Free Mode)

유료 API 없이 매일 사업 아이템 인사이트를 뽑기 위한 운영 가이드.

## 목표
- 매일 오전 9시, 실행 가능한 인사이트 1개
- 국내 3개 + 해외 3개 출처
- 출처/수집일/근거/확신도/검증 액션 포함

## 소스 풀

### 국내 (3)
1. 네이버 데이터랩 (검색 트렌드)
2. KOSIS / 공공 리포트 (시장 통계)
3. 네이버 카페/유튜브 댓글/국내 커뮤니티 (문제 발화)

### 해외 (3)
1. Google Trends (검색 트렌드)
2. Reddit (r/smallbusiness, r/Entrepreneur, r/marketing)
3. 업계 리포트 공개 페이지 (HubSpot, McKinsey, Deloitte 등)

## 일일 루틴 (45분)
1. 수집 20분
   - 국내 10개 신호 + 해외 10개 신호
2. 분류 10분
   - 고통 강도(1~5)
   - 지불 가능성(1~5)
   - 실행 가능성(1~5)
3. 선택 5분
   - 상위 1개 인사이트 확정
4. 검증 설계 10분
   - 오늘 실행할 1개 액션 정의

## 스코어링 규칙
- 총점 = 고통 강도 + 지불 가능성 + 실행 가능성 (최대 15)
- 11점 이상: 즉시 테스트
- 8~10점: 보류 후 추가 데이터
- 7점 이하: 백로그

## 리포트 포맷 (고정)
1) 인사이트 한 줄
2) 근거 데이터
3) 출처
   - [국내-문제발화] URL (수집일)
   - [국내-수요지표] URL (수집일)
   - [국내-경쟁대체재] URL (수집일)
   - [해외-문제발화] URL (수집일)
   - [해외-수요지표] URL (수집일)
   - [해외-경쟁대체재] URL (수집일)
4) 확신도 (높음/중간/낮음)
5) 오늘 검증 액션 1개 (30~60분)

## 신뢰도 등급
- A: 공공통계/원문조사/실제 전환 데이터
- B: 대형 리서치/플랫폼 공식 리포트
- C: 커뮤니티 정성 데이터

## 주의사항
- robots.txt/이용약관 준수
- 개인정보 수집 금지
- 과도한 요청/우회 접근 금지
- 커뮤니티 데이터는 가설용(C등급)으로만 사용
